• HTML (HyperText Markup Language) is the standard language used to create web pages. It structures content using elements like headings, paragraphs, images, and links. HTML forms the basic skeleton of a website and works with CSS and JavaScript.

  • CSS (Cascading Style Sheets) is used to style and design web pages. It controls the layout, colors, fonts, spacing, and overall appearance of HTML elements. CSS works with HTML to make websites visually attractive and responsive.

  • JavaScript is a programming language used to make web pages interactive and dynamic. It allows you to add features like animations, form validation, and user interactions. JavaScript works with HTML and CSS to create fully functional websites

  • React is a JavaScript library used to build user interfaces (UI), especially for web applications. It allows developers to create reusable components and update the UI efficiently. React is mainly used for building fast, interactive, and dynamic front-end applications.

  • Node.js is a JavaScript runtime environment that allows you to run JavaScript outside the browser. It is used to build backend applications like servers, APIs, and real-time apps. Node.js is fast and efficient because it uses an event-driven, non-blocking architecture.
