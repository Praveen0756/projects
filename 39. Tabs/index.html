HTML

HTML (HyperText Markup Language) is the standard language used to create web pages. It structures content using elements like headings, paragraphs, and links. It works with CSS and JavaScript to design and add interactivity to websites

CSS

CSS is used to style web pages and control their layout. It works with HTML to design elements like colors, fonts, and spacing. It improves the overall look and user experience of a website.

JavaScript

JavaScript is a programming language used in web development. It adds interactivity and dynamic behavior to web pages. It works with HTML and CSS to create complete websites.

React.Js

React is a JavaScript library for building user interfaces. It uses components to create reusable parts of a webpage. It helps build fast and interactive web applications.

Node.Js

Node.js is a server-side runtime for executing JavaScript code. It is used to build backend services and APIs. It enables fast and scalable web applications.
